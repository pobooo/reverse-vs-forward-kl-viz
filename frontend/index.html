说明

Reverse KL = Eq[log q − log p]：q 落在 p 为零的地方会被无穷惩罚， 所以 q 会退缩到 p 的某个 mode 上（mode-seeking / zero-forcing）。

Forward KL = Ep[log p − log q]：q 在 p 有质量的地方却是零会被无穷惩罚， 所以 q 会尽量覆盖所有 mode（mode-covering / zero-avoiding）。

参考: blog.evjang.com/2016/08/variational-bayes.html

公式说明

目标分布 p(x)（固定，3 个 mode） $$p(x)=\sum_{k=1}^{K_p} w_k\,\mathcal{N}(x\mid \mu_k,\sigma_k^2)$$
代码中 $\mu_k$ 在 $[-4\frac{K_p-1}{2},\;+4\frac{K_p-1}{2}]$ 等距取值, $\sigma_k=0.6$, $w_k=1/K_p$。
变分分布 q(x; θ)（可训练，1 个分量） $$q(x;\theta)=\sum_{j=1}^{K_q}\pi_j\,\mathcal{N}(x\mid m_j,s_j^2)$$
$\theta=\{m_j,\;s_j,\;\pi_j\}$ 通过 Adam 优化, $\pi_j=\mathrm{softmax}(\text{logits})_j$。
Reverse KL mode-seeking $$D_{\mathrm{KL}}(q\|p)=\int q(x)\log\frac{q(x)}{p(x)}\,dx$$
q 落在 p 为零的位置会被无穷惩罚（zero-forcing）, 所以 q 会收缩到某个 mode 上。
Forward KL mode-covering $$D_{\mathrm{KL}}(p\|q)=\int p(x)\log\frac{p(x)}{q(x)}\,dx$$
p 有质量但 q 为零的位置会被无穷惩罚（zero-avoiding）, 所以 q 会尽量覆盖所有 mode。
实现：两种 KL 都通过 数值积分（梯形法则，2000 点密网格）在 PyTorch 里求得, 梯度对 $\theta$ 自动回传。无采样噪声，$\hat D_{\mathrm{KL}}\ge 0$ 严格成立。
⚠ 实际拟合更复杂：mode-seeking / mode-covering 是倾向性描述，不是确定性保证。 最终结果由 K_q 容量、学习率、初始化、迭代步数、局部最优 共同决定。 例如 K_q ≥ K_p 时两种 KL 结果几乎一样；K_q < K_p 时 reverse KL 也常出现"多个分量挤同一 mode"的局部最优。 真实 VI/LLM 蒸馏还会叠加采样噪声、正则项、模型架构等因素——本 demo 是建立直觉用的极简案例。
